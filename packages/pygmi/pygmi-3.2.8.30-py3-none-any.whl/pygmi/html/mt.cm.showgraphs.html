Show Graphs¶

This displays graphs of the station data.

  1. The apparent resistivity versus period for the selected elements.

  2. Phase versus period for the selected elements.

  3. Tipper magnitude versus period for the selected elements.

  4. Tipper angle versus period for the selected elements.

  5. Station Name – If data for multiple stations were imported the dropdown list can be used to select a specific station

  6. Graph Type – This can be either ‘xy, yx’ or ‘xx, yy’.

  7. Values for a point in any of the graphs over which the mouse pointer is hovering.

  8. Standard image display settings that allow the user to zoom into specific areas of the image, move the zoomed in area around, return to the full image, save the image, etc.
