Layer Stack and ResamplingÂ¶

This will allow datasets to be merged into a multiband dataset and resampled to a common cell size. It can also be used for a single dataset to be resampled. The resulting rows and columns of the new grid are displayed.

The interface has the following options and information:

  1. Cell Size - This represents the size of a square raster grid cell, in the units of the grid (normally metres).

  2. The number of rows and columns of the resampled grid.

  3. Common mask for all bands - This option forces the nodata areas to be the same on all bands. When no common mask is applied, the two datasets retain their original coverage. When a common mask is applied, both data sets will have the same nodata areas.
