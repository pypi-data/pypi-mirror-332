======================
GeoThibaud's admin site
======================

.. module:: thibaud.contrib.gis.admin
    :synopsis: GeoThibaud's extensions to the admin site.

``GISModelAdmin``
=================

.. class:: GISModelAdmin

    .. attribute:: gis_widget

        The widget class to be used for
        :class:`~thibaud.contrib.gis.db.models.GeometryField`. Defaults to
        :class:`~thibaud.contrib.gis.forms.widgets.OSMWidget`.

    .. attribute:: gis_widget_kwargs

        The keyword arguments that would be passed to the :attr:`gis_widget`.
        Defaults to an empty dictionary.
