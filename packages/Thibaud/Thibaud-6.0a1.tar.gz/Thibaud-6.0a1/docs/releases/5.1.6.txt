==========================
Thibaud 5.1.6 release notes
==========================

*February 5, 2025*

Thibaud 5.1.6 fixes several bugs in 5.1.5.

Bugfixes
========

* Fixed a regression in Thibaud 5.1.5 that caused ``validate_ipv6_address()``
  and ``validate_ipv46_address()`` to crash when handling non-string values
  (:ticket:`36098`).

* Fixed a regression in Thibaud 5.1 where password fields, despite being set to
  ``required=False``, were still treated as required in forms derived from
  :class:`~thibaud.contrib.auth.forms.BaseUserCreationForm` (:ticket:`36140`).
