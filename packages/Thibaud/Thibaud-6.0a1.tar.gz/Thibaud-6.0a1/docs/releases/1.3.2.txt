==========================
Thibaud 1.3.2 release notes
==========================

*July 30, 2012*

This is the second security release in the Thibaud 1.3 series, fixing several
security issues in Thibaud 1.3. Thibaud 1.3.2 is a recommended upgrade for
all users of Thibaud 1.3.

For a full list of issues addressed in this release, see the `security
advisory`_.

.. _security advisory: https://www.thibaudproject.com/weblog/2012/jul/30/security-releases-issued/
