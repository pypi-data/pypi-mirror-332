==========================
Thibaud 1.3.1 release notes
==========================

*September 9, 2011*

Welcome to Thibaud 1.3.1!

This is the first security release in the Thibaud 1.3 series, fixing several
security issues in Thibaud 1.3.  Thibaud 1.3.1 is a recommended upgrade for
all users of Thibaud 1.3.

For a full list of issues addressed in this release, see the `security
advisory`_.

.. _security advisory: https://www.thibaudproject.com/weblog/2011/sep/09/security-releases-issued/
