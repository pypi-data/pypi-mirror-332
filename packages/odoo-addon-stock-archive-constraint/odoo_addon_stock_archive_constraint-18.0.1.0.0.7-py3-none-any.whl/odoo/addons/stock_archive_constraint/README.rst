Prevents archiving products or storage locations that have existing
inventory or related transactions.

**Table of contents**

.. contents::
   :local:

Usage
=====

1. Go to Settings > Users > Edit a user and check the "Manage Multiple
   Stock Locations" permission
2. Go to Inventory > Configuration > Configuration > Settings, in the
   Warehouse section and check the "Storage Locations" option
3. Go to Inventory > Configuration > Locations and attempt to archive a
   location that has an associated stock move, either as the source or
   destination.
4. Go to inventory > Products > Products and try to archive a product
   associated with the stock selection, which is in stock, or which has
   some movement assigned to it
